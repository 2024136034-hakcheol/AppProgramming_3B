# PRD Step 2 — 기능 강화

## 목표
사용성 향상 및 레시피 품질 개선 (Step 1 완료 후 진행)

## 추가 기능

| 기능 | 설명 |
|------|------|
| 재료 수동 편집 | 인식된 재료 목록에서 항목 추가/삭제 후 재검색 |
| 레시피 필터 | 요리 종류(한식/중식/양식/일식), 조리 시간, 난이도 필터 |
| 레시피 상세 보기 | 영양 정보, 칼로리 예상치 포함 |
| Rate Limit 처리 UI | 재시도 중 진행 상황을 스피너/메시지로 표시 |
| 결과 공유 | 레시피 결과를 텍스트로 클립보드 복사 |

## UI 개선
- 로딩 애니메이션 (API 응답 대기 중)
- 모바일 반응형 레이아웃
- 재료 태그(chip) UI로 직관적 편집
