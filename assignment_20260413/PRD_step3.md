# PRD Step 3 — 고도화

## 목표
개인화 및 지속적 사용 유도 (Step 2 완료 후 진행)

## 추가 기능

| 기능 | 설명 |
|------|------|
| 히스토리 | 이전 분석 기록 저장 (LocalStorage 또는 SQLite) |
| 즐겨찾기 | 마음에 드는 레시피 북마크 |
| 냉장고 관리 | 재료를 직접 등록/관리하는 가상 냉장고 기능 |
| 쇼핑 목록 | 레시피에 필요하지만 없는 재료 자동 추출 → 쇼핑 목록 생성 |
| 다국어 지원 | 레시피 출력 언어 선택 (한국어/영어/일본어) |

## 기술적 개선
- 이미지 압축 전처리 (업로드 전 리사이즈로 API 비용 절감)
- 결과 캐싱 (동일 재료 조합은 재요청 없이 캐시 반환)
- 모델 선택 UI (사용자가 Vision/LLM 모델을 직접 선택 가능)
